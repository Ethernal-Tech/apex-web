# Apex Bridge Web API

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## How to create migration

```bash
$ npm --name=layerzero1 run migration:generate
```

## How to update database

```bash
$ npm run migration:run
```
