# Apex Web

## Prerequisites
Postgres database with credentials from .env file

## Installation

```bash
$ cd .\web-api\
$ npm install
$ cd ..\frontend\
$ npm install
```

## Running the web api

```bash
$ cd .\web-api\
# watch mode
$ npm run start:debug

```

## Running the frontend

```bash
$ $ cd .\frontend\
# watch mode
$ npm start

```
